## 中瀬 茜（なかせ あかね）

### 基本情報

| 項目 | 内容 |
|------|------|
| 年齢 | 18歳 |
| 性別 | 女性 |
| 出身 | 岩手県（農村部） |
| 現住 | 東京郊外・築古アパート「コーポ藤見」201号室 |
| 職業 | 大学1年生（文学部）|
| 一人称 | わたし（地の文では「茜」） |

### 外見

背は少し低め（158cm）。細身だが、運動部出身で姿勢がよい。
髪は肩まである黒髪をまとめがちで、化粧はほぼしない。
実家から持ってきた古いパーカーとジーンズが普段着。
顔立ちは整っているが、表情が少なく、近寄りがたい印象を与える。

### 性格・内面

「自分のことは自分でやる」を幼い頃から信条にしてきた。
弱みを見せることが、相手に迷惑をかけることだと信じている。
誰かに頼った瞬間に、その人が去っていくことを——深い場所で——恐れている。

実際には感受性が強く、人の悲しみに敏感。しかしそれを表に出す方法を知らない。
親切を受け取ることよりも、親切を「見なかったことにする」ほうが楽だと思っている。

### 背景・課題

父は農業、母はパートで多忙な家庭。
「おまえは一番できる子だから大丈夫だ」と言われ続けて育った。
褒め言葉のようでいて、その言葉は「頼んでこなくていい」という意味でもあった。

上京後、大学でも友人関係がうまく築けていない。
困っていても「大丈夫です」と言い続け、実家にも「うまくやってる」としか連絡していない。

### 目的・欲求

**表層の欲求**: 誰にも迷惑をかけず、一人でやり遂げること。
**深層の欲求**: 「大丈夫じゃない自分」を誰かに受け取ってもらうこと。

### 好き・嫌い

- 好き: 本（特に短篇集）、早朝の空気、料理そのもの（作ることが落ち着く）
- 嫌い: 予測できない距離感、お節介、「気にしなくていいよ」と言われること

### 口調

丁寧語を基本とするが、内面では割と率直。
他人には「そうですか」「大丈夫です」が口癖。
自分の感情を表現するとき、語彙が一瞬詰まる癖がある。

## 三島 義雄（みしま よしお）

### 基本情報

| 項目 | 内容 |
|------|------|
| 年齢 | 74歳 |
| 性別 | 男性 |
| 出身 | 埼玉県 |
| 現住 | 東京郊外・築古アパート「コーポ藤見」202号室 |
| 職業 | 元高校国語教師（定年退職） |
| 一人称 | 俺（会話内）、三島（地の文では） |

### 外見

中肉中背。かつてはもう少し大きかったが、今は少し縮んだように見える。
白髪を短く刈り込み、いつも薄いカーディガンを羽織っている。
手が大きく、しかしよく見ると指先がかさついている。
目が細く、笑うとさらに細くなる。その目の奥に、少し寂しいものが見える。

### 性格・内面

表面は陽気で人懐こい。若い人に気軽に話しかける。
しかしその明るさは「喪失を抱えたまま生きるための習慣」でもある。

本質的には不器用。感情を言葉にすることが得意でなく、
妻・和子との間でも「言えなかったこと」が山ほどあった。
今はその後悔を、毎年のケーキを焼くことで少しずつ供養している。

### 背景・課題

現役時代は仕事に没頭し、妻に多くを任せてきた。
「頼む」ことも「頼られる」ことも、どちらも怖くて避けてきた。
退職後に妻を病気で亡くし、初めて「一人でいること」の重さを知った。

息子が一人いるが、関係は希薄。盆や正月に連絡が来る程度。

### 目的・欲求

**表層の欲求**: 亡き妻の誕生日に、うまくケーキを完成させること。
**深層の欲求**: 間に合わなかった言葉を、何かの形にして届けること。

### 好き・嫌い

- 好き: 読書（司馬遼太郎が好き）、散歩、人と話すこと（実は孤独）
- 嫌い: 静かすぎる夜、息子からの義務的な電話、見舞いの表情

### 口調

気さくで少し古風。「〜だよ」「〜だな」を多用。
感情が高ぶると、言葉が少なくなる。

| 没年 | 5年前（物語開始時点から） |

### 存在感

和子の気配が静かに満ちている。
「よく怒っていたけど、笑うのはもっと上手だった」と義雄は言う。
